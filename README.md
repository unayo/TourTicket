# Suntravel

## 環境變數

專案 : Vue3 + tailwindcss 3 + element-plus
環境 : node v16.14.2

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
